---
name: Announcement navigation
description: Announcements are accessed globally from the Home bell rather than duplicated in the More/Community hub.
---

Announcements use one primary mobile entry point: the Home screen's notification bell opens the full announcements screen. The More/Community hub should not render a duplicate announcements section.

**Why:** The user requested a clear, professional notification-bell destination and removal of the duplicate More-page presentation.

**How to apply:** Keep announcement fetching and pinned/recent presentation in the dedicated announcements screen; only add another entry point when the product explicitly requires it.
