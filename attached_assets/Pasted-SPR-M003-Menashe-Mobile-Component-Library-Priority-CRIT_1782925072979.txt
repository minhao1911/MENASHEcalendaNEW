------------------------------------------------------------
MISSION
------------------------------------------------------------

Build the complete reusable Menashe Mobile Component Library.

This is NOT a screen-building sprint.

This sprint creates the reusable UI components that every future
mobile screen will use.

All components MUST consume the Menashe Mobile Design Language
(MMDL) created in SPR-M002.

No hardcoded styles.

No duplicated UI.

No business logic.

Presentation only.

------------------------------------------------------------
NON-NEGOTIABLE RULES
------------------------------------------------------------

DO NOT

❌ Modify any Web UI

❌ Modify Web Components

❌ Modify Web Navigation

❌ Modify Existing Business Logic

❌ Build Home Screen

❌ Build Calendar Screen

❌ Build Sanctuary Screen

❌ Build AI Screen

❌ Duplicate shared components

Everything belongs ONLY inside

src/mobile/

The Web Application remains production frozen.
