# MENASHE PLATFORM

## ENGINEERING SPRINT INSTRUCTION

### SPR-021 — Production Validation & Release Readiness

ROLE

Act as a Senior QA Engineer, Release Engineer, Security Tester, Product Tester, Accessibility Tester, and Mobile Tester.

Do NOT add new features.

Do NOT redesign the UI.

Do NOT change the database schema.

Do NOT create new APIs.

Your only goal is to prove that the Menashe Platform is ready for Version 1.0 production.

* Offline
* Empty search
* Invalid URL

* API timeout

No crashes.
