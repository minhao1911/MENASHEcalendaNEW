------------------------------------------------------------
MISSION
------------------------------------------------------------

Build the Menashe Mobile Shell (MOS).

The Mobile Shell becomes the permanent framework that hosts every
future screen.

The Shell is NOT a feature.

It is the operating environment of the application.

Every screen built afterwards must live inside this Shell.
